Hello Rorigami

Rorigami (Rocket Origami) is a small, fast, easily extendable, easy to use static website template, batteries included. Knowing the most basic of HTML and Markdown is all you really need. It uses the popular Bulma CSS framework so when opting for HTML, you are left with plenty of options to work with

No builds required

That's right, no need to build anything, simply add content, and push your changes. Nothing more, nothing less.

But why?

Glad you asked. In recent years, having something static which doesn't require "building" or even worse - "compiling" has been treated as heresy. As a result, the internet has been overflooded with thousands and thousands of tools, templating engines, building pipelines which not only do make the whole process slower and more counter-productive but they come with carrying an additional weight - setting up, learning how they all work and how to use them. Which in addition, makes your life even harder if you are on the move. Rorigami is trying to address that issue with plain simplicity. Make a change, save and move on.
